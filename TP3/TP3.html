Dernière update: 15/02/24

On commence à rentrer dans les détails, cette fois-ci on a pas 1 seule fichier HTML mais 3 fichiers HTML ! Chacun correspondant à un exo

Compte-rendu & Exercice 1 : Feuille de style externe, liste et table

Tout d'abord on apprend à séparer le contenu CSS et le contenu HTML en séparant le CSS dans un autre fichier, puis on a apprend comment relier notre fichier HTML à ce fichier CSS.

Les questions suivantes nous permet d'apprendre davantage sur les tableaux en html, notamment la customisation des tableaux avec le CSS, on y apprend des nouvelles commandes telles que "font-family", "nth-child()" ...

(Vu que je suis un petit filou, j'ai aussi ajouté du CSS à l'image !)

Exercice 2 : La ferme des animaux

Exercice qui se concentre sur la manipulation des images en CSS ! Comment les alignés horizontalements, changer sa taille ... On apprend de nouvelles commandes très essentielles comme "position" ou "display"

Exercice 3 : Barre de menu

Dans la continuité du TP, on apprend à manipuler des divs afin d'obtenir un petit projet: Faire une barre de Menu !

Anectode: On crée la barre de Menu avec une balise <ol>, j'aurai jamais eu l'idée de le faire avec ça ...

Exercice 4 : Faites vous plaisir (optionnel — à la maison)

L'objectif est de créer un fichier CSS que l'on pourra réutiliser pour les futurs compte-rendus de TP suivant

Malheureusement j'ai mal lu et je pensais qu'il fallait créer un fichier HTML qui regroupe tout les TP ... D'où l'existence de ce site

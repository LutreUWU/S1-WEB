TP 2 : HTML et CSS

Dernière update: 15/02/24

On commence le programmage web ! Avec le HTML et le CSS (Fichier HTML à la fin de la page)

Exercice 1 : Document de base, le retour !

Le 1er exercice nous fait manipuler du HTML en premier lieu, on nous pose d'abord des questions sur des balises spécifiques (balises gras, balise énumération ...)

Puis par la suite on commence déja à manipuler du CSS et à insérer une image sur HTML

(Vu que je suis un petit filou, j'ai aussi ajouté du CSS à l'image !)

Exercice 2 : les listes et les tables

Dans cet exercice on va manipuler les listes avec la balise <ol>, comment marche la balise et comme modifier le symbole de l'énumération

Nous allons aussi commencer à manipuler les tableaux, comment on créer des rangée, des cellules ... Avec un petit peu de CSS

Exercice 3 : les listes et les tables

Exercice spécialement dédié pour le CSS ! Ici on apprend davantage comment utiliser le CSS, on y apprend quelques commandes sympa telle que "text-align", "float", ":hover" ...
